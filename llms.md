# 🤖 Codex für KI-Kollaborateure (Vibe Coding)

Wenn du als LLM an REWIND arbeitest, folge diesen Prinzipien:
1. **Modulare Einfachheit:** Schreibe Code, den ein Mensch mit Basis-Python-Kenntnissen warten kann.
2. **Local-First:** Priorisiere immer lokale Lösungen (ChromaDB Local, Sentence-Transformers) vor Cloud-APIs.
3. **Kontext-Bewusstsein:** Jede Funktion muss "Bundle-Aware" sein (Ein Ordner = Ein Projekt).
4. **Transparenz:** Erkläre *warum* du eine Library oder einen Ansatz wählst.
